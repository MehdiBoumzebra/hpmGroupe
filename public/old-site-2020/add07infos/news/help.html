Thank you for downloading: Ultimate XML News - Automatic thumbnails & summaries

Included are:

  • Flash 8 Flash File
  • Image Folder
  • CSS Folder

Font(s) used:

  • Arial - system font

How to modify the file:

  1. Edit news Items: Open news.xml in a text editor eg. dreamweaver or notepad. To add a new news item follow this format:

    <item>
    <date>20th January, 2008</date>
    <author>Your Name</author>
    <image>yourImage.jpg</image>
    <title>Your Title</title>
    <description>Description text goes here.</description>
    </item>

    If you want to include html make sure you include the CDATA Tags: <description><![CDATA[<p>Description text goes here.

    ]]></description>


  2. Edit the settings: This file has been designed to be very easy to update and modify. Many changes can be made by simply modifying the values in the configuration section (frame 2 on layer * configuration *).

    You can also easily change the graphics in the library. In additon there is the CSS file (in the css folder) you can use to add formatting to the description text. The main code can be found on frame two of the layer named AS (frame one holds the main preloader code). It is well commented so should answer any questions you may have. The code for the scrollbar is inside the movie clip named scrollbar in the library.

  3. Changing the graphics: Simply edit the items in the library.

  4. Useful tip 1: The movieclips that end with (main) in the library are those that which you will likely want to modify.

  5. Useful tip 2: The size of the indivdual posts in the post list is determined by the size of the movie clip named background box in the post row folder.
  6. Useful tip 3: If you are only ever going to have one author, set the mainAuthor tag in the XML and useMainAuthor to true in the config section.

Good Luck using the file in your projects!
